# GitHub User Finder

## API Used
GitHub Users API

https://api.github.com/users/{username}

## Features
- Search GitHub users
- Fetch live API data
- Async/Await used
- DOM Manipulation
- Error Handling
- Loading Spinner

## Files
- index.html
- style.css
- app.js

## How to Run
1. Download all files
2. Keep them in one folder
3. Open index.html in browser
